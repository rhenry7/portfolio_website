About


Veghead is an application I developed using React, JavaScript, HTML/CSS and Adobe CC. The app connects to the Yelp. This site was deployed using Github. The goal of this project was to allow vegan and vegetarian users an easier and friendlier experience for finding restaurants catering to their specific diets. As a health conscious eater, and an advocate for healthy lifestyles overall, I enjoyed creating an application that will help users achieve their goals.

Technologies Used


Languages: HTML/CSS/JavaScript
Frameworks: React
Other: Yelp API
Deployment: Github pages
